# SwimJournal

**Лабораторна робота №2**
*Дисципліна:* Веб-розробка ПЗ
*Тема:* Створення структури веб-проєкту

## Опис проєкту
SwimJournal — це прототип веб-застосунку для плавців-аматорів і спортсменів. Проєкт поєднує функції трекера результатів і особистого щоденника тренувань, допомагаючи користувачам відстежувати свій прогрес та залишатися вмотивованими.

## Структура проєкту
* **index.html** — Головна сторінка з описом сервісу.
* **dashboard.html** — Особистий кабінет користувача (таблиця тренувань).
* **contacts.html** — Сторінка зворотного зв'язку з формою.
* **login.html** — Сторінка входу/реєстрації.
* **css/style.css** — Стилізація сторінок згідно з брендбуком.
* **images/** — Графічні матеріали (логотип, банери).

## Технології
* **HTML5** (семантична верстка)
* **CSS3** (стилізація, Flexbox)
